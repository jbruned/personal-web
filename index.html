Investigación sobre Active Learning

En mi Trabajo de Fin de Grado, titulado A review of Active Learning methods for classification problems, investigué sobre el estado del arte de Active Learning.

Active Learning es una técnica aplicable a Machine Learning en situaciones donde no se dispone de suficientes datos etiquetados, cuyo objetivo es optimizar el proceso de obtención de etiquetas.

Tecnologías utilizadas:
Python, SciKit-Learn, pyTorch, MatPlotLib, NumPy, Pandas, LaTeX...

Instalación de red local en nave industrial

Desarrollo de app para Android con servidor REST

También como parte del grado universitario, he desarrollado una aplicación para Android de forma individual. La aplicación, llamada FitTastic, permite al usuario llevar un seguimiento de la ingesta y el consumo de calorías de forma sencilla e intuitiva.

Implementación de un dispositivo Arduino para controlar videojuegos de conducción

Como parte de la asignatura Sistemas Multimedia y Diseño Centrado en el Usuario, desarrollé un dispositivo basado en una placa Arduino que permite controlar videojuegos de conducción de una forma más sencilla y divertida.

El dispositivo está compuesto por unos pedales impresos en 3D que se conectan por USB, y una aplicación para Android que utiliza el giroscopio para controlar el volante a través de una conexión WiFi.

En este proyecto se ha tenido especialmente en cuenta la usabilidad, accesibilidad, etc. La interfaz de usuario se ha desarrollado en Processing utilizando Java como lenguaje de programación.

El proyecto destaca por tener un fuerte componente de hardware y electrónica, ya que utiliza numerosos componentes como, por ejemplo, transistores.

Tecnologías, lenguajes y entornos utilizados:
Arduino, TinkerCad (circuitos y 3D), Processing, Arduino IDE, oscP5 (comunicación en red), serial monitor (comunicación con Arduino por USB), Android Studio, Java...

Explotación de un dataset desbalanceado mediante el proceso KDD

Este proyecto, realizado de forma grupal con otros dos compañeros como parte de la asignatura Minería de Datos, consistió en la explotación de un dataset desbalanceado para predicción de ictus.

Para llevar a cabo esta tarea, aplicamos distintas fases del proceso conocido como KDD (Knowledge Discovery from Databases). Cabe destacar:

  • Imputación de valores perdidos mediante modelos entrenados con el resto de variables
  • Detección y tratamiento de outliers
  • Normalización de datos
  • Instance Selection para hacer frente al problema del desbalanceo: concretamente implementé de una variante de RMHC que incrementó el f-score drásticamente
  • Optimización de hiperparámetros
  • Integración en una pipeline
  • Análisis de la importancia de las variables

Implementación de juegos distribuidos en red

De forma individual, desarrollé dos aplicaciones de juegos distribuidas en red, utilizando una arquitectura cliente-servidor.

  • En primer lugar, desarrollé utilizando Java y el IDE NetBeans una sala de juegos distribuida en red, con una interfaz de usuario (GUI) basada en Java Swing. Esta aplicación destaca por la gran abstracción que permite implementar nuevos juegos de forma muy sencilla tan solo implementando los métodos pertinentes de interfaces y clases abstractas.
  • Por otro lado, implementé un parchís en C que permite a los jugadores conectarse al servidor desde sus casas e iniciar partidas conforme se vayan conectando otros jugadores. Cabe destacar que programé este parchís desde cero, gestionando todo el envío de mensajes a través de buffers y tareas similares, con una estructura de código modularizada e intuitiva, emulando clases. Así mismo, diseñé una interfaz que permitía ver el tablero utilizando solo carácteres ASCII.
